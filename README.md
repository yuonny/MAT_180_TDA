# MAT180_TDA

This is the github repo for David Kim's MAT 180 Project "Topological Data Analysis Application: Comparing Gram Negative and Positive Gnome Distributions."

Currently, the code is set up for general use of anyone who wants to compute Betti Curves of their data. My current set up loads the dataset as a matrix of stacked Pytorch tensors which are then converted into numpy. If your data is already in a numpy format, then such an initial step can be ignored.
